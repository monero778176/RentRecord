# RentRecord
### 製作目的
繳了快兩年的房租，在租屋時光逐漸尾聲時 <br>
才發現可以透過自身所學來實作一個快速紀錄的登記project <br>
我必須在月初的時候快速的統計我們室友的房租、以及個人的用電和分配的公費 <br>
每一次都是機械式的工作 <br>
因此才有了這次的小project <br>

### Python版本要求
python 3.8 <br>
